# Calculatrice Basique en C
Une petite calculatrice en console supporte addition, soustraction, multiplication et division.

## Fonctionnalités
- Addition (`+`)
- Soustraction (`-`)
- Multiplication (`*`)
- Division (`/`) avec gestion de la division par zéro

## Compilation 
```bash
gcc calculatrice.c -o calculatrice

## Exécution
./calculatrice
